# Initialize Vite Project

## Context

Create the base React + TypeScript project using Vite. This is the foundation for all subsequent work.

## Acceptance Criteria

- [ ] Project created with `npm create vite@latest` using React + TypeScript template
- [ ] `npm run dev` starts the development server successfully
- [ ] `npm run build` produces a production build without errors
- [ ] ESLint configured for TypeScript
- [ ] Clean up default Vite boilerplate (remove demo content from App.tsx)

## Out of Scope

- Adding any application-specific code
- Installing additional dependencies beyond what Vite provides
- Setting up testing framework

## Notes

Use strict TypeScript settings (`"strict": true` in tsconfig).
