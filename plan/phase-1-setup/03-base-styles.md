# Base Styles & Layout Shell

## Context

Create the foundational CSS and layout structure. The app uses a two-panel layout: agent list on the left, main content area on the right.

## Acceptance Criteria

- [ ] Reset default browser styles (minimal reset, not a full framework)
- [ ] Define CSS variables for:
  - Colors for available/unavailable/neutral states
  - Spacing scale
  - Font settings
- [ ] Create two-panel layout structure:
  - Left sidebar (~250px fixed width)
  - Right main area (flexible)
- [ ] Add placeholder content to verify layout works
- [ ] Layout should fill viewport height

## Out of Scope

- Responsive/mobile layouts
- Dark mode
- Animation/transitions
- Component-specific styling

## Notes

Use plain CSS or CSS modules. Avoid adding a CSS framework - keep dependencies minimal.
